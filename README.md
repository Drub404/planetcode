# Planet: Code Discord Bot (Archived)

This repository contains the source code for the **Planet: Code** Discord bot.

Planet: Code was a community-focused Discord server created to bring programmers together.  
The bot was designed to **automate server management**, provide **useful utilities**, and include **fun interactive features and games** for members.

This project is no longer active and is archived for reference and learning purposes.

---

## Project purpose
- Build a community-focused Discord bot
- Automate moderation and server workflows
- Create interactive commands and mini-games
- Experiment with larger-scale bot logic
- Learn how different systems interact in a single project

---

## Technologies used
- **Skript**
